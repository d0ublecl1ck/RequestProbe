## ADDED Requirements

### Requirement: 用户可以批量下载请求包
系统 MUST 允许用户在请求监听列表中勾选一个或多个请求记录，并将这些请求包批量下载到当前任务目录。下载后的请求包 MUST 以结构化文件形式保存，且同一请求记录重复下载时 MUST 避免重复写入。

#### Scenario: 下载选中的请求包
- **WHEN** 用户在请求列表中勾选一个或多个请求并点击下载
- **THEN** 系统 MUST 将这些请求包保存到当前任务目录中的请求下载位置
- **THEN** 每个成功下载的请求记录 MUST 更新下载状态和下载路径

#### Scenario: 未勾选请求时下载
- **WHEN** 用户未勾选任何请求就点击下载
- **THEN** 系统 MUST 阻止下载动作
- **THEN** 系统 MUST 返回明确提示

#### Scenario: 已存在文件时重复下载请求包
- **WHEN** 用户再次下载已经落盘的请求记录
- **THEN** 系统 MUST 不重复写入相同文件
- **THEN** 该请求记录 MUST 保持已下载状态
