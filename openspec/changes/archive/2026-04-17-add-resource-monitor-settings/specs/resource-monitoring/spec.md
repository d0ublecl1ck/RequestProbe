## ADDED Requirements

### Requirement: 用户可以配置资源监听默认保存位置

系统 MUST 提供一个可持久化的资源监听默认保存根目录设置，并允许用户在设置页面查看、修改和恢复该目录。资源监听新任务启动时 MUST 在该根目录下为任务 UUID 创建独立子目录。

#### Scenario: 查看当前默认保存位置

- **WHEN** 用户打开设置页面
- **THEN** 系统 MUST 展示当前资源监听默认保存根目录
- **THEN** 若用户尚未自定义过设置，系统 MUST 展示内置默认目录

#### Scenario: 修改默认保存位置

- **WHEN** 用户在设置页面输入或选择一个有效目录并保存
- **THEN** 系统 MUST 持久化该目录作为新的资源监听默认保存根目录
- **THEN** 系统 MUST 在后续新建任务时使用该目录作为根目录

#### Scenario: 恢复默认保存位置

- **WHEN** 用户点击恢复默认值
- **THEN** 系统 MUST 将资源监听默认保存根目录重置为应用内置默认目录

#### Scenario: 启动任务时沿用默认保存位置

- **WHEN** 用户在资源监听页面启动新的监听任务
- **THEN** 系统 MUST 在默认保存根目录下创建 `<taskId>/` 子目录
- **THEN** 当前任务的下载目录 MUST 指向该子目录
