## MODIFIED Requirements

### Requirement: 用户可以通过右侧 Tab 分栏查看字段探针结果

系统 MUST 将字段探针页面右侧结果区组织为固定 Tab 分栏，而不是将四类结果长期纵向堆叠在同一滚动列中。结果区 MUST 提供当前分组的清晰标签和可感知的切换状态，以便用户在“请求代码与测试”和“测试摘要与简化代码”之间快速往返。

#### Scenario: 用户切换查看不同结果分组

- **WHEN** 用户点击任一右侧结果分组 Tab
- **THEN** 系统 MUST 立即切换到对应结果分组面板
- **THEN** 当前分组 MUST 通过明显的激活视觉反馈与未选中分组区分
- **THEN** 已切换后的内容 MUST 继续保留原有结果、空状态提示和操作按钮
