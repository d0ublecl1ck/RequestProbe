## MODIFIED Requirements

### Requirement: 桌面端必须通过左侧侧边栏切换工作区

桌面端界面 MUST 使用左侧侧边栏承载全局工作区导航，而不是顶部 Tabs。侧边栏 MUST 至少提供“字段探针”“资源监听”和“设置”入口，并以明确的激活态展示当前工作区。侧边栏 SHOULD 同时展示当前工作台名称或简短说明，以强化用户对当前产品上下文的感知。

#### Scenario: 进入主界面

- **WHEN** 用户打开主界面
- **THEN** 左侧 MUST 展示统一的工作台导航壳层
- **THEN** 当前激活工作区 MUST 通过明显高亮与其他入口区分
- **THEN** 用户 MUST 可以直接切换到“字段探针”“资源监听”或“设置”
